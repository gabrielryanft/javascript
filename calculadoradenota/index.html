Fórmula usada para calcular a nota:

Legenda: Letras que representam os trimestres na fórmula:

  • primeiro trimestre é representado pela letra: p;
  • segundo trimestre é representado pela letra: s;
  • terceiro trimestre (Nota necessária para passar) é representado pela letra: t.

  ( 60   -   ( 3 p   +   3 s ) ) 4   =   t

Evite usar exatamente as mesmas notas apresentadas ao calcular sua média. Lembre-se de que O EscolaRS PODE ARREDONDAR AS NOTAS PARA CIMA.
